## redis 插件使用说明

    1、无需引用无需定义 直接使用 redis 变量即可
    
    2、使用示例
        set方法：redis.set('0', 'test', '123');
        成功后 返回 字符串 OK 失败 返回 null
        get方法：redis.get('0', 'test');
        成功返回对应的键值  失败返回 null
       
    3、异步调用 必须 async await
